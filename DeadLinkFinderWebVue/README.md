# GitHub Repository Link Checker

A modern Vue 3 application that checks for dead links in GitHub repository README files.

## Features

- Search for GitHub users/organizations
- Automatically checks the 5 most recently updated repositories
- Displays repository statistics (stars, forks, last updated)
- Shows link status (OK, Warning, Bad) with HTTP status codes
- Filter links by status
- Modern, responsive UI

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Development

The application is built with:
- Vue 3
- TypeScript
- Vite
- Octokit (GitHub API client)
